## ✦ What This App Does

- **Ask questions** in **English, Urdu, or Hindi** — auto-detected
- Returns **authentic hadith** from the 14 Infallibles with full **Arabic + English + Urdu + Hindi** translations
- **Source citations** from priority Akhbari books (Kitab Sulaim, Mizan al-Hikmah, Tuhaf al-Uqool)
- **Explicitly rejects** Usuli marja fatwas, ijtihad, and clerical authority when queried
- **Daily Hadith** button for random wisdom
- 100% free — runs on Streamlit Community Cloud + HF Free Tier

## 🚀 Deployment: Streamlit Community Cloud (FREE)

### Step 1: Fork / Clone This Repo

```bash
# Clone locally or fork on GitHub
git clone https://github.com/YOUR_USERNAME/sayanjali-wisdom-14.git
cd sayanjali-wisdom-14
```

### Step 2: Get a Hugging Face API Token (Free)

1. Go to [huggingface.co](https://huggingface.co) → Sign up (free)
2. Navigate to **Settings → Access Tokens**
3. Click **New Token** → Type: `Read` → Name it `sayanjali`
4. Copy the token (starts with `hf_...`)

> The free HF Inference API allows access to Qwen QwQ-32B and DeepSeek models.

### Step 3: Deploy on Streamlit Community Cloud

1. Go to [share.streamlit.io](https://share.streamlit.io)
2. Sign in with GitHub
3. Click **New app**
4. Select your repo, branch `main`, file `app.py`
5. Click **Advanced settings** → **Secrets**
6. Paste:
   ```toml
   HF_TOKEN = "hf_your_token_here"
   ```
7. Click **Deploy!** — your app will be live in ~2 minutes

### Step 4: (Optional) Local Development

```bash
# Install dependencies
pip install -r requirements.txt

# Create secrets file
mkdir -p .streamlit
echo 'HF_TOKEN = "hf_your_token_here"' > .streamlit/secrets.toml

# Run
streamlit run app.py
```

## ☪ Akhbari Shia Position

This platform adheres strictly to the **Akhbari** school of Shia Islam:

- ✅ Accepts: Quran + authenticated hadith of the 14 Infallibles
- ❌ Rejects: Marja-following (taqlid), ijtihad, Usuli jurisprudence
- ❌ Will NOT cite: Sistani, Khamenei, Shirazi, or any other marja/moulvi
- If asked about Usuli fatwas → explicitly states the Akhbari rejection and redirects to Imam hadith
